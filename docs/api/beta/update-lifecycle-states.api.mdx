<MethodEndpoint method={"patch"} path={"/identity-profiles/{identity-profile-id}/lifecycle-states/{lifecycle-state-id}"}></MethodEndpoint>



This API updates individual lifecycle state fields using the [JSON Patch](https://tools.ietf.org/html/rfc6902) standard.

A token with ORG_ADMIN or API authority is required to call this API.
