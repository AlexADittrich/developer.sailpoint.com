<h1 className={"openapi__heading"}>Remove members from Governance Group</h1>

<MethodEndpoint method={"post"} path={"/workgroups/{workgroupId}/members/bulk-delete"}></MethodEndpoint>



This API removes one or more  members from a Governance Group.  A token with API, ORG_ADMIN authority is required to call this API.

>  **Following field of Identity is an optional field in the request.**

>  **name**
