<h1 className={"openapi__heading"}>Get identities for a suggested entitlement (for a role)</h1>

<MethodEndpoint method={"get"} path={"/role-insights/{insightId}/entitlement-changes/{entitlementId}/identities"}></MethodEndpoint>



Role insights suggests entitlements to be added for a role. This endpoint returns a list of identities in the role, with or without the entitlements, for a suggested entitlement so that the user can see which identities would be affected if the suggested entitlement were to be added to the role.
