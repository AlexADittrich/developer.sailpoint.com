<h1 className={"openapi__heading"}>Patch a Governance Group</h1>

<MethodEndpoint method={"patch"} path={"/workgroups/{id}"}></MethodEndpoint>



This API updates an existing governance group by ID. 
The following fields and objects are patchable:
  * name
  * description
  * owner

A token with API or ORG_ADMIN authority is required to call this API.
