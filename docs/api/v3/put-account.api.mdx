<h1 className={"openapi__heading"}>Update Account</h1>

<MethodEndpoint method={"put"} path={"/accounts/{id}"}></MethodEndpoint>



Use this API to update an account with a PUT request. 
This endpoint submits an account update task and returns the task ID. 
A token with ORG_ADMIN authority is required to call this API.
>**NOTE: You can only use this PUT endpoint to update accounts from sources of the "DelimitedFile" type.**
