<h1 className={"openapi__heading"}>Set Lifecycle State</h1>

<MethodEndpoint method={"post"} path={"/identities/{identity-id}/set-lifecycle-state"}></MethodEndpoint>



This endpoint will set/update an identity's lifecycle state to the one provided and updates the corresponding Identity Profile.
A token with ORG_ADMIN or API authority is required to call this API.
