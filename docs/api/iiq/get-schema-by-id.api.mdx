<h1 className={"openapi__heading"}>Returns a Schema based on id.</h1>

<MethodEndpoint method={"get"} path={"/Schemas/{schemaId}"}></MethodEndpoint>



The Schema with the matching id is returned. The id is the URN of the SCIM resource. The 'attributes' field holds the schema-specific attributes which differ depending on Schema type.
